# AI 语音助手

一个基于SMT自研大模型SMITTY智能语音交互助手，支持语音识别和语音合成，为用户提供自然、便捷的对话体验。

## 🌟 功能特点

- 🎙️ **语音识别**：支持实时语音输入，自动转换为文字
- 🔊 **语音合成**：AI 回答自动转换为语音输出
- 💬 **智能对话**：接入星火 API，提供智能对话服务
- 📝 **对话历史**：支持查看历史对话记录
- 🎯 **即时反馈**：提供 Toast 提示，操作更清晰

## 🛠️ 技术架构

### 核心模块

- **语音服务** (`voice-service.js`)
  - 语音识别初始化与控制
  - 语音合成及参数调节
  - 实时语音反馈

- **对话处理** (`app.js`)
  - 用户输入处理
  - 消息历史管理
  - UI 状态控制

- **AI 通信** (`Smittyapi.js`)
  - WebSocket 连接管理
  - 消息发送与接收
  - 认证处理

### 界面组件

- 响应式布局
- 动态状态显示
- 侧边栏历史记录
- 交互式控制按钮
